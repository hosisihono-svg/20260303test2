日報タスク分析ツール

日報から「自分のタスクの見える化」と「完了率の感触」をすぐに確認できます。

v1.1
for Daily Report Template
How to 左に日報テキストを貼り付けて「解析する」を押すだけで、右側にタスク一覧と完了状況が並びます。

1. 日報テキストを貼り付け

Input

日報テンプレに沿ったテキスト（1日分でも複数日分でもOK）を、このテキストエリアに貼り付けてください。 特に「### 2) 今日やったこと」配下の箇条書きからタスクを抽出します。

貼り付けた日報はブラウザ内だけで処理されます。サーバ送信は行いません。

2. 解析結果

Overview

日付 ラベル ステータス 内容

まだタスクが抽出されていません。左側に日報テキストを貼り付けて「解析する」を押してください。

日報テンプレの「### 2) 今日やったこと」配下の行（例: - **(A) タスク名**: 説明）からタスク情報を抽出しています。

完了 / 進行中 などのキーワードから、ざっくりとしたステータスも自動判定します。
